## Color Scheme

### Gradients
- **Primary**: Blue (#2563eb) to Purple (#9333ea)
- **Success**: Green (#16a34a) to Teal (#14b8a6)
- **Warning**: Orange (#ea580c) to Amber (#f59e0b)

### Status Colors
- **Approved**: Green background (#dcfce7), Green text (#166534)
- **Pending**: Yellow background (#fef9c3), Yellow text (#854d0e)
- **Rejected**: Red background (#fee2e2), Red text (#991b1b)

### UI Elements
- **White Cards**: #ffffff with shadow
- **Gray Background**: #f9fafb
- **Text Primary**: #111827
- **Text Secondary**: #6b7280
- **Borders**: #e5e7eb

## Responsive Behavior

### Desktop (>1024px)
- Full sidebar visible
- Multi-column layouts
- Large cards and spacing

### Tablet (768px - 1024px)
- Collapsible sidebar
- 2-column layouts
- Medium cards

### Mobile (<768px)
- Hidden sidebar (hamburger menu)
- Single column layouts
- Stacked cards
- Full-width buttons

## Interactions

### Hover Effects
- Cards: Slight elevation (shadow increase)
- Buttons: Color darken + scale(1.05)
- Links: Color change

### Click Feedback
- Buttons: Brief scale down then up
- Forms: Focus ring (blue)
- Navigation: Smooth transitions

### Loading States
- Skeleton screens for content
- Spinner for actions
- Progress bars for uploads

This visual guide represents the complete FinTrack application interface!
